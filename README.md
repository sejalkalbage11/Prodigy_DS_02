# Prodigy_DS_02

## Task 2: Data Cleaning and Exploratory Data Analysis (EDA)

### Objective
Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset to identify patterns, relationships, and trends.

### Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

### Dataset
Titanic Dataset

### Steps Performed
- Imported the dataset
- Checked missing values
- Cleaned the data
- Performed exploratory data analysis
- Created multiple visualizations
- Observed relationships between variables

### Visualizations
- Survival Count
- Survival by Gender
- Survival by Passenger Class
- Age Distribution
- Fare Distribution
- Age Box Plot
- Correlation Heatmap

### Conclusion
- Female passengers had a higher survival rate.
- First-class passengers had better survival chances.
- Most passengers were between 20–40 years old.
- Passenger class and fare influenced survival.
